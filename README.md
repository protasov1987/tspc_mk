# Codex Project

Базовый шаблон фронтенд-проекта (HTML + CSS + JS), настроенный для работы с GitHub и CI (GitHub Actions).

## Структура

- src/ — исходники (HTML, CSS, JS)
- .github/workflows/ci.yml — проверка проекта на GitHub Actions
- package.json — npm-скрипты для разработки и тестов

## Быстрый старт

1. Установить зависимости:
   npm install

2. Запустить режим разработки:
   npm run dev

3. Открыть в браузере адрес, который покажет live-server (обычно http://127.0.0.1:8080).
