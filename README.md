# Codex Project

Базовый шаблон фронтенд-проекта (HTML + CSS + JS) с простым сервером на Express и локальной базой SQLite для сохранения сообщений. Настроен для работы с GitHub и CI (GitHub Actions).

## Структура

- src/ — исходники (HTML, CSS, JS)
- .github/workflows/ci.yml — проверка проекта на GitHub Actions
- package.json — npm-скрипты для разработки и тестов

## Быстрый старт

1. Установить зависимости:
   npm install

2. Запустить сервер (Express + SQLite + статика из `src`):
   npm start

   Сервер слушает порт `3000` по умолчанию. Его можно переопределить переменной `PORT`.

3. Открыть в браузере `http://localhost:3000`.

4. (Опционально) Для чисто статической разработки без API можно использовать:
   npm run dev
