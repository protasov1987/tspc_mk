# Codex Project

Базовый шаблон фронтенд-проекта (HTML + CSS + JS) с лёгким сервером на Node.js и локальным хранением сообщений в файле. Настроен для работы с GitHub и CI (GitHub Actions).

## Структура

- src/ — исходники (HTML, CSS, JS)
- .github/workflows/ci.yml — проверка проекта на GitHub Actions
- package.json — npm-скрипты для разработки и тестов

## Быстрый старт

1. Установок не требуется — сервер использует только встроенные модули Node.js.

2. Запустить сервер со статикой и API:
   npm start

   Сервер слушает порт `3000` по умолчанию. Его можно переопределить переменной `PORT`.

3. Открыть в браузере `http://localhost:3000`.

4. (Опционально) Скрипт `npm run dev` идентичен `npm start` и запускает тот же сервер.
