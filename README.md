# Маршрут технологических карт (статический сайт)

Лёгкое веб-приложение на чистых HTML/CSS/JS для ведения технологических карт и маршрутов операций. Все данные хранятся в браузере (localStorage), поэтому сайт полностью работает на статическом хостинге.

## Содержимое репозитория
- `src/` — страница (`index.html`), стили (`style.css`) и клиентская логика (`main.js`).
- `.github/workflows/pages.yml` — публикация статики из `src` на GitHub Pages.
- `.github/workflows/ci.yml` — минимальная проверка (`npm test`).
- `package.json` — метаданные проекта (без зависимостей).

## Как открыть локально
1. Откройте файл `src/index.html` в браузере **или** поднимите любой статический сервер (например, `python -m http.server 3000 -d src`).
2. Интерфейс сразу готов к работе: создавайте техкарты, маршруты, используйте поиск и печать штрихкодов.

## Публикация на GitHub Pages
1. Создайте репозиторий на GitHub и загрузите в него файлы (ветка `main`).
2. Убедитесь, что включён GitHub Actions и присутствует workflow `.github/workflows/pages.yml`.
3. После пуша откройте адрес: `https://<username>.github.io/<repo>/` (дождитесь завершения Actions).

## Замечания
- Серверная часть не требуется — приложение работает целиком в браузере.
- При очистке кэша браузера локальные данные будут удалены.
